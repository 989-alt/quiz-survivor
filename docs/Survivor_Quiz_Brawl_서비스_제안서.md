# Survivor Quiz Brawl 서비스 제안서

## 단원별 수학 학습 서바이벌 게임

이천송정초등학교 교사 이선학

---

## 1. 제안 개요

### 1.1. 제안 배경 및 목적

| 구분 | 내용 |
| --- | --- |
| 제안 배경 | 초등 수학은 단원별 핵심 개념을 반복적으로 풀이해 익혀야 정착되지만, 학생들은 동일한 형식의 학습지·문제집을 반복하는 데 쉽게 흥미를 잃습니다. ① 교사는 단원이 끝날 때마다 형성평가·복습 활동을 별도로 준비해야 하고, ② 빠르게 푸는 상위권 학생과 시간이 더 필요한 하위권 학생이 한 교실에서 같은 속도로 진행되며, ③ 학생들이 자율적으로 반복 학습할 동기를 갖기 어렵고, ④ Kahoot·퀴즐렛 등 기존 학습 게임은 "단순 퀴즈 풀이" 형식이라 게임으로서의 재미가 부족해 학생들이 금방 식상해합니다. 또한 학생들에게 인기 있는 모바일 게임(뱀파이어 서바이버 류)은 교육 콘텐츠와 무관하여, 게임 시간이 곧 학습 손실로 이어집니다. |
| 제안 목적 | 본 제안은 학생들에게 인기 있는 **뱀파이어 서바이버 형식의 액션 게임**과 **22개정 교육과정 아이스크림미디어 수학 5~6학년 단원별 4지선다 퀴즈**를 결합한 학습 게임 「**Survivor Quiz Brawl**」을 아이스크림 미디어 교수학습지원 플랫폼의 정식 콘텐츠로 채택할 것을 제안합니다. 이를 통해 ① 학생 스스로 단원별 퀴즈를 반복 풀이하도록 자연스러운 동기를 부여하고(레벨업 게이트로 퀴즈 정답이 곧 게임 진행), ② 단원별 통합 랭킹을 통해 학급·학년·전국 단위의 건전한 경쟁을 유도하며, ③ 교사의 형성평가·복습 활동 준비 부담을 0에 가깝게 줄이고, ④ 22개정 아이스크림미디어 교과서 단원 구성 그대로 정합성을 보장하는 것을 목표로 합니다. |

### 1.2. 서비스 이용 대상 범위

초/중/고등학교 교사를 대상으로 하는 수업 지원 디지털 콘텐츠 및 서비스 플랫폼 개발 및 제공

- **1차 타깃**: 초등학교 5~6학년 담임교사 (현재 5·6학년 × 1·2학기 × 6단원 × 15문제 = **360문제 매핑 완료**)
- **2차 타깃**: 초등학교 1~4학년 담임교사 (저학년은 알고리즘 자동생성 + 큐레이션 혼합 적용)
- **3차 타깃**: 중·고등학교 수학 교사 (단원 데이터 확장 후)
- **부가 활용**: 자율·동아리(창체) 시간, 단원평가 직전 복습, 방과후·방학 자율학습, 학급 단위 학습 이벤트(예: "이번 주 1단원 챔피언")

## 2. 아이디어 소개

### 2.1. 서비스명 및 컨셉

- **콘텐츠/서비스 명**: Survivor Quiz Brawl (서바이버 퀴즈 브롤)
- **해결하려는 문제**:
  - 단원별 반복 풀이가 필수인 초등 수학에서 학생들의 학습 동기와 흥미를 지속적으로 유지하기 어렵다.
  - Kahoot·퀴즐렛 등 기존 학습 게임은 단순 퀴즈 풀이 형식이라 게임적 몰입이 약해 학생들이 빠르게 식상해진다.
  - 일반 모바일 게임은 교육 콘텐츠가 전혀 없어, 학생들이 게임에 쓰는 시간이 곧 학습 시간 손실로 이어진다.
  - 교사가 단원이 끝날 때마다 별도의 형성평가·복습 활동을 직접 준비해야 하는 부담이 크다.
  - 한 교실 안에서 학습 속도 격차가 큰 학생들이 동일 페이스로 진행되어 상·하위권 모두 이상적인 도전 수준을 받지 못한다.
- **해결 방법**: 학생들에게 친숙한 **뱀파이어 서바이버 형식의 탑다운 액션 게임 메커니즘**(끝없이 몰려오는 몬스터, 무기 강화, 레벨업, 보스 웨이브)에 **22개정 아이스크림미디어 수학 단원별 4지선다 퀴즈**를 **레벨업 게이트**로 결합했습니다. 학생은 학년 → 학기 → 단원을 선택해 게임에 입장하고, 게임 중 레벨업이 발생할 때마다 해당 단원의 퀴즈 1문항이 출제됩니다. **정답 시** 무기 강화 카드 3종 중 1종을 선택하며 게임을 계속하고, **오답 시** 레벨이 1단계 취소되며 누적 XP가 0으로 초기화되는 페널티가 부여됩니다. 게임 종료 후 점수는 `score + survivalTime×10 + level×100`의 가중점수로 자동 환산되어 **단원별 통합 랭킹**(Firestore 기반 Top 100)에 등재되며, 학생은 학급·전국 단위로 본인 순위를 즉시 확인할 수 있습니다.

### 2.2. 서비스 주요 기능 소개

**기능 1 — 22개정 아이스크림미디어 수학 단원 매핑 (5~6학년, 360문제)**
학년 → 학기 → 단원의 4단계 트리에서 단원을 선택하면, 해당 단원에 큐레이션된 4지선다 15문제가 매 게임 새로 셔플되어 출제됩니다. 현재 **5학년 1·2학기 + 6학년 1·2학기 각 6단원, 총 24단원 × 15문제 = 360문제**가 22개정 교육과정 아이스크림미디어 교과서 단원 구성과 1:1 매핑되어 있으며, 단원의 핵심 개념·계산 패턴·실생활 응용을 균형 있게 다루도록 설계되었습니다. (예: 5-1 1단원 「자연수의 혼합 계산」 → 괄호·곱셈·나눗셈 우선순위 / 5-1 2단원 「약수와 배수」 → 최대공약수·최소공배수 / 6-2 5단원 「원의 넓이」 → 원주율·반지름·지름 응용)

**기능 2 — 뱀서라이크 게임 메커니즘 (50+ 학용품 테마 무기, 보스 웨이브)**
검증된 「뱀파이어 서바이버」 스타일의 탑다운 액션 게임을 Phaser 3 엔진으로 구현했습니다. 플레이어는 8방향(WASD·방향키·모바일 가상 조이스틱)으로 이동하며, 자동 발사되는 무기로 끝없이 몰려오는 몬스터를 처치합니다. 무기는 **연필·자·바나나 발사기·도토리·종이비행기·도시락·물풍선·햄스터·나비** 등 학생 친화적인 50종 이상의 학용품·자연 테마 무기로 구성되어 있으며, 각 무기는 8레벨까지 강화 가능합니다. 30초마다 새 웨이브가 시작되고, 매 3웨이브마다 **보스**가 등장합니다. 몬스터는 일반·탱커(HP 3배)·빠른형(HP 0.4배)으로 차별화되어 있으며, 웨이브 진행에 따라 HP·이동속도가 가파르게 증가합니다(`HP = 15 + wave×6 + wave²×0.4`).

**기능 3 — 레벨업 퀴즈 게이트 (정답=강화, 오답=페널티)**
플레이어가 처치한 몬스터의 XP 젬을 모아 레벨업이 발생할 때마다 게임이 일시 정지되고 해당 단원의 4지선다 퀴즈가 1문항 출제됩니다. **정답 시** 점수 +50점 보너스와 함께 무기 강화 카드 3종 중 1종을 선택하며 게임을 계속하고, **오답 시** 레벨이 1단계 취소되며 누적 XP가 0으로 초기화됩니다. 한 번에 여러 레벨업이 누적될 경우 모든 레벨업마다 퀴즈가 순차 출제되어, 학생이 학습을 우회할 수 없도록 설계되었습니다. 정답·오답 시 별도의 효과음과 시각 피드백이 즉시 제공되며, 풀이는 무한 반복 셔플로 동일 단원 내 다양한 문항을 접할 수 있습니다.

**기능 4 — Firebase 단원별 통합 랭킹 (닉네임 + 익명 인증)**
게임 종료 후 점수는 `점수 + 생존시간(초)×10 + 레벨×100`의 가중점수로 자동 환산되어 Firestore에 등록됩니다. 별도 회원가입 없이 익명 인증(`signInAnonymously`)으로 처리되며, 학생은 닉네임 12자만 입력하면 됩니다. 랭킹은 **단원별로 분리**(`/leaderboards/{unitId}/scores`)되어 같은 단원을 학습한 학생들끼리만 경쟁하며, **Top 100 리스트와 본인 등수**가 즉시 강조 표시됩니다. Firestore 보안 규칙으로 `score ≤ 1,000,000`, `survivalTime ≤ 7200초`, 닉네임 1~12자 등 어뷰징 방어가 적용되어 있습니다.

**기능 5 — 픽셀아트 판타지 게임 자산 (배경·BGM·SFX·UI)**
Google Gemini 2.5 Flash Image (Nano Banana 2)로 생성한 **2048×2048 픽셀아트 마법 숲 배경**, Pixabay 라이선스의 **판타지 BGM**, 몬스터 피격·처치 / 플레이어 피격을 톤별로 차별화한 **SFX 4종**, 그리고 보라·시안·금빛으로 통일된 **다크 판타지 UI**를 갖추고 있습니다. 게임 캔버스 위에는 보라·시안 안개 layer 8개가 부유하며 타일 이음매를 자연스럽게 가립니다. 모든 자산은 라이선스가 명확한 무료·CC0/Pixabay 자료로 구성되어 상업 배포에 제약이 없습니다.

**기능 6 — 멀티 디바이스 지원 (PC·모바일·태블릿)**
PC 키보드(WASD·방향키)와 모바일 가상 조이스틱을 모두 지원하며, Tailwind CSS 4의 반응형 레이아웃으로 1280×720 데스크톱부터 375px 폭 모바일까지 매끄럽게 대응합니다. 별도 앱 설치 없이 **웹 브라우저로 즉시 플레이** 가능하여 학교 컴퓨터실·태블릿·가정 학습 모두 지원합니다.

### 2.3. 구현 방안

**기술적 접근 방법**

- **프론트엔드**: React 19 + TypeScript + Vite 7 + Tailwind CSS 4
- **게임 엔진**: Phaser 3.90 (탑다운 액션, 물리 충돌, tileSprite 무한 배경, Tween/SFX 시스템)
- **상태 관리**: Zustand (게임/퀴즈 상태)
- **백엔드**: Firebase (Firestore + Authentication 익명 로그인) — 별도 서버 0
- **단원 데이터**: TypeScript 정적 모듈(`src/data/units/grade5-1.ts` 등) — 빌드 타임에 번들에 포함되어 오프라인 지원
- **AI 생성 자산**: Google Gemini 2.5 Flash Image (Nano Banana 2) — 픽셀아트 마법 숲 배경
- **사운드 자산**: Pixabay (라이선스 명확)
- **배포**: GitHub Pages (`gh-pages` 브랜치 자동 배포, `npm run deploy` 한 줄)
- **환경 변수**: `VITE_FIREBASE_*` 6종 (정식 채택 시 아이스크림 미디어 게이트웨이 키 관리로 전환)
- **CI/CD**: gh-pages 패키지로 GitHub Actions 없이도 단일 명령으로 배포

**개발 로드맵 (제안)**

| 단계 | 기간 | 주요 활동 |
| --- | --- | --- |
| 1단계: 내부 심사 | 제안 후 1개월 이내 | MVP를 아이스크림 미디어 내부에서 시연·심사. 5~6학년 담임교사 자문단 5~10명 대상 사전 시범 사용 후 피드백 수집 (단원별 문제 난이도·게임 밸런스·학생 수업 적용성) |
| 2단계: 피드백 반영 | 1~2개월 | 아이스크림 미디어 브랜드 가이드라인 적용 (UI/UX 리스킨 — 마법 숲 → 아이스크림미디어 캐릭터 테마 등 옵션), 교수학습지원 플랫폼 SSO 연동, 운영자 관리 콘솔 개발(랭킹 모더레이션·문제 추가 인터페이스), 모바일 가상 조이스틱 UX 개선 |
| 3단계: 콘텐츠 확장 | 2~4개월 | 1~4학년 수학 + 중·고 수학 단원 확장. 국어·사회·과학 등 타 교과 문제은행 추가. 단원당 15문제 → 30문제 확장. 저학년은 알고리즘 자동생성 (예: `8 + 5 = ?`) + 큐레이션 혼합. 교과서 PDF → 단원 메타데이터 자동 추출 파이프라인 구축 |
| 4단계: 베타 운영 | 2~3개월 | 시범 학교 30~50개교 베타 운영, 사용 로그 분석(단원별 평균 정답률·평균 생존시간·랭킹 진입률), 게임 밸런스 미세 조정, 어뷰징 방어 강화, 학급 단위 랭킹·학교 단위 랭킹 추가 |
| 5단계: 정식 출시 | 베타 종료 후 | 아이스크림 미디어 교수학습지원 플랫폼 정식 콘텐츠로 등재, 전국 회원 교사 대상 오픈, 시즌 단위 이벤트 운영(단원별 챔피언, 학급 토너먼트) |

**확장 방안 (정식 출시 이후)**

- **교사 대시보드**: 학급 학생들의 단원별 평균 정답률·반복 학습 횟수·취약 단원 자동 분석 리포트
- **학급/학년 단위 랭킹**: 학교·학급 코드로 분리된 별도 랭킹 보드
- **시즌 이벤트**: 매월 새 단원 챔피언, 단원 마스터리 배지
- **AI 오답 노트**: 학생이 틀린 문제 자동 누적 → 다음 게임에서 우선 출제
- **학부모 모드**: 자녀의 단원별 학습 누적 시간·정답률 리포트
- **PWA 변환**: 오프라인 플레이 지원, 홈 화면 추가 가능

## 3. 세부 구성 요소

### 3.1. 서비스 화면 (스토리보드)

아래 화면은 https://989-alt.github.io/survivor-quiz-brawl/ 에서 직접 캡처한 실제 동작 화면을 토대로 구성되었습니다. 정식 채택 시 아이스크림 미디어 브랜드 가이드라인에 맞춰 리스킨됩니다.

**[화면 1] 홈 — 판타지 배경 + 2개 진입 카드**

- 배경: AI 생성 픽셀아트 마법 숲 + 보라/시안 오로라 + 별 파티클
- 메인 카드 2종: **시작하기** (학년·단원 선택) / **랭킹 보기** (단원별 Top 100)
- 푸터: "© 2026 Survivor Quiz Brawl · 22개정 교육과정 · 5~6학년 수학 · Phaser 3 + React + Firebase"
- [캡처 위치: 홈 첫 화면 전체]

**[화면 2] 학년 → 학기 → 단원 → 닉네임 4단계 선택 흐름**

- 진행 인디케이터: 4개 도트가 단계별 채워짐
- 1단계 학년: 5학년 / 6학년 (대형 카드)
- 2단계 학기: 1학기 / 2학기
- 3단계 단원: 6개 카드 (단원번호·제목·설명·문제 수 표시. 예: "1단원 자연수의 혼합 계산 · 15문제")
- 4단계 닉네임: 선택한 단원 미리보기 + 닉네임 입력 (12자 제한, localStorage 자동 기억)
- [캡처 위치: 단원 선택 화면 — 6개 카드가 모두 보이는 상태]

**[화면 3] 게임 화면 — HUD + 픽셀 마법 숲 + 캐릭터·몬스터**

- 좌상단 HUD: Lv.1 / Wave 1 / HP 100/100 / XP 0/25 (보라/시안 진행바)
- 우상단 HUD: 점수 / kills / 생존시간
- 게임 캔버스: 2048×2048 마법 숲 배경 위에서 캐릭터가 8방향 이동, 자동 무기 발사, 몬스터 처치 시 데미지 숫자(`-15`) 띄우기, XP 젬 드롭
- 모바일: 우하단 가상 조이스틱
- [캡처 위치: 웨이브 3~5 진행 중, 무기 2~3개 강화된 상태]

**[화면 4] 레벨업 퀴즈 오버레이**

- 게임이 어둡게 페이드 → 중앙에 4지선다 카드 등장
- 단원명·문제번호 표시 (예: "5학년 1학기 1단원 · 자연수의 혼합 계산")
- 15초 타이머 (보라 진행바)
- 4개 옵션 (A·B·C·D 라벨, 색상 구분)
- 답안 선택 시 즉시 정답/오답 표시 + 해설 → 2초 후 자동 진행
- [캡처 위치: 퀴즈 출제 직후 + 답안 선택 후 결과 표시 상태]

**[화면 5] 무기 업그레이드 선택 (3장 카드)**

- 정답 후 등장하는 3장의 강화 카드
- 각 카드: 무기 아이콘 + 한글 이름 (바나나 발사기·연필·자 등) + 현재 레벨 → 다음 레벨 효과 (+데미지 / +개수 / +관통 등)
- 새 무기 신규 획득 / 기존 무기 강화 / 진화 가능 시 진화 카드 우선 노출
- [캡처 위치: 3장 카드가 모두 보이는 상태]

**[화면 6] 게임오버 + 점수 자동 등록**

- 가중점수 강조 표시 (예: `2,470`)
- 4분할 통계 카드: 점수 / 생존 / 레벨 / 처치
- 자동 Firestore 등록 상태 표시 ("✓ 랭킹 등록 완료")
- 3개 버튼: **랭킹 / 다시 하기 / 홈으로**
- [캡처 위치: 점수 등록 완료 직후]

**[화면 7] 단원별 통합 랭킹 (Top 100 + 내 등수)**

- 학년·학기·단원 필터 (드롭다운 + 토글)
- 본인 최고 기록 카드 (시안 강조, "Top 100 N위" 표시)
- Top 100 리스트: 1위 금색·2위 은색·3위 동색 강조, 본인 row 시안 배경
- 각 row: 순위 / 닉네임 / 가중점수 / 점수·생존시간·레벨·처치 세부
- [캡처 위치: 5학년 1학기 1단원 랭킹 — 본인이 Top 50 내 진입한 상태]

※ 위 화면 캡처는 제안서 제출 시 실제 스크린샷으로 대체 또는 첨부될 예정입니다. 동작 흐름은 https://989-alt.github.io/survivor-quiz-brawl/ 에서 실시간으로 확인 가능합니다.

### 3.2. 사용자 흐름 (User Flow)

```
[홈 진입]
  → [시작하기 클릭]
  → [학년 선택 (5/6)]
  → [학기 선택 (1/2)]
  → [단원 선택 (6개 중 1)]
  → [닉네임 입력 (localStorage 자동 기억)]
  → [게임 시작 — Phaser 캔버스 + BGM 재생]
  → [몬스터 처치 → XP 젬 획득 → 누적]
  → [레벨업 발생 → 게임 일시정지 → 4지선다 퀴즈 출제]
       ├── [정답] → 무기 강화 카드 3장 → 1장 선택 → 게임 재개 (+50점)
       └── [오답] → 레벨 1단계 취소 + XP 0으로 초기화 → 게임 재개
  → [반복: 30초 웨이브 / 매 3웨이브마다 보스 / 무한 진행]
  → [HP 0 도달 → 게임오버]
  → [가중점수 자동 환산 + Firestore 등록]
  → [PostGameOverlay: 랭킹 / 다시 하기 / 홈으로]
       ├── [랭킹] → 단원별 Top 100 + 본인 등수
       ├── [다시 하기] → 같은 단원으로 새 게임
       └── [홈으로] → 홈 화면
```

## 4. 기대 효과

### 4.1. 정성적 효과

- **교사의 형성평가·복습 활동 준비 시간 절감**: 단원이 끝날 때마다 별도 학습지·문제집·Kahoot 세트를 준비하던 1~2시간을 0에 가깝게 단축 (학생에게 단원만 안내하면 됨)
- **학생의 자율 반복 학습 동기 강화**: 게임 재미 자체가 학습 동기가 되어, 교사의 별도 독려 없이 학생이 스스로 같은 단원을 반복 풀이 (단원당 15문제 무한 셔플)
- **단원 정합성 보장**: 22개정 아이스크림미디어 교과서 단원과 1:1 매핑되어 진도와 형성평가 사이 어긋남 0
- **개별화된 학습 페이스**: 빠른 학생은 더 오래 생존하며 더 많은 문제를 풀고, 느린 학생은 본인 페이스로 진행하면서 같은 단원을 반복 학습
- **건전한 경쟁 문화**: 학급·전국 단위 단원별 랭킹으로 학생들이 상위권을 목표로 자발적 추가 학습
- **모바일 게임 시간을 학습 시간으로 전환**: 학생들에게 친숙한 뱀서라이크 게임 메커니즘으로 게임 시간이 곧 학습 시간이 되도록 유도
- **2022 개정 교육과정의 디지털 소양 함양**: 디지털 매체를 통한 자기주도 학습·문제해결 역량 강화
- **수업 외 시간 활용도 극대화**: 자율·동아리 시간, 방과후, 가정 학습, 단원평가 직전 복습 등 다양한 맥락에서 즉시 활용

**핵심 차별 포인트 요약**

「Survivor Quiz Brawl」은 ① **검증된 뱀파이어 서바이버 게임 메커니즘**과 **22개정 아이스크림미디어 수학 단원 360문제**를 **레벨업 퀴즈 게이트**로 결합하여 학생이 게임을 즐기는 시간이 곧 단원 반복 학습 시간이 되도록 설계되었으며, ② **단원별 Firebase 통합 랭킹**으로 학급·전국 단위 건전한 경쟁을 유도하고, ③ **별도 회원가입·앱 설치 없이** 웹 브라우저로 즉시 플레이 가능하며, ④ **단원 데이터 모듈을 추가하기만 하면** 1~4학년·중·고·타 교과로 자유롭게 확장 가능한 구조입니다. 이 4가지 결합은 Kahoot·퀴즐렛 같은 단순 퀴즈, 일반 학습지 PDF, 일반 모바일 게임 어디에서도 제공되지 않는 차별 포인트입니다.

  1. **시작하기 → 5학년 → 1학기 → 1단원 「자연수의 혼합 계산」 → 닉네임 입력 → 게임 시작** — 레벨업 시 출제되는 4지선다 퀴즈 흐름 체험
  2. 정답·오답 페널티 직접 경험 (오답 시 레벨 1단계 취소 + XP 0 초기화)
  3. 게임오버 후 **랭킹** 버튼 클릭 → 본인 점수가 Firestore에 자동 등록되고 단원별 Top 100에서 본인 위치 확인
  4. **홈 → 랭킹 보기** → 학년·학기·단원 필터로 다른 단원 랭킹 비교 (예: 5-1 1단원 vs 6-1 4단원 「비와 비율」)
  5. **다시 하기**로 같은 단원 반복 플레이 — 문제 셔플 확인
